# pre-test-voca-game
This is a repository for the pre-test back end developer of the Voca game, on behalf of Dhewan Agum Mahendra.

## How to run
Run this code with go command:
```bash
go run main.go
```

## Command
Command for use the application

Create parking lot with specific capacity
```bash
create_parking_lot {capacity}
```

Park a car with specific car number
```bash
park {car_number}
```

Leave a car from parking lot
```bash
leave {car_number}
```

Get parking lot status
```bash
status
```

Exit from application
```bash
exit
```
